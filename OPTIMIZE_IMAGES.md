# Инструкция по оптимизации изображений

## Проблема
Изображения на сайте слишком большие (13+ MB), что замедляет загрузку страницы и ухудшает показатели PageSpeed Insights.

## Решение

### 1. Установите зависимости

```bash
npm install
```

Это установит библиотеку `sharp` для конвертации изображений в WebP формат.

### 2. Запустите скрипт оптимизации

```bash
npm run optimize-images
```

или

```bash
node optimize-images.js
```

### 3. Что делает скрипт

Скрипт автоматически:
- Конвертирует изображения в WebP формат (на 70-90% меньше размер)
- Создает несколько размеров для адаптивных изображений:
  - `hero-slider-1.jpg` → `hero-slider-1-1920.webp`, `hero-slider-1-1280.webp`, `hero-slider-1-960.webp`, `hero-slider-1-640.webp`
  - `event-2.PNG` → `event-2.webp`, `event-2-750.webp`, `event-2-500.webp`
  - `menu/1.png`, `2.png`, `3.png` → WebP версии с разными размерами
  - `special-dish-banner.png` → WebP версии с разными размерами

### 4. HTML уже обновлен

HTML файлы уже обновлены для использования:
- `<picture>` элементов с WebP источниками
- `srcset` для адаптивных изображений
- Fallback на оригинальные форматы для старых браузеров

### 5. Ожидаемый результат

После оптимизации:
- **hero-slider-1.jpg**: 5.7 MB → ~800 KB (экономия ~4.8 MB)
- **event-2.PNG**: 2.2 MB → ~300 KB (экономия ~2.2 MB)
- **menu изображения**: 1.2-1.4 MB → ~200 KB каждое (экономия ~1 MB каждое)
- **special-dish-banner.png**: 1.3 MB → ~200 KB (экономия ~1.2 MB)

**Общая экономия: ~12 MB** (с 13+ MB до ~1-2 MB)

### 6. Проверка

После запуска скрипта:
1. Проверьте, что WebP файлы созданы в папках `assets/images/`
2. Откройте сайт в браузере и убедитесь, что изображения загружаются
3. Проверьте PageSpeed Insights снова - показатели должны значительно улучшиться

### 7. Если что-то пошло не так

Если WebP файлы не созданы, но HTML обновлен:
- Браузер автоматически использует оригинальные файлы (fallback)
- Сайт будет работать, но без оптимизации
- Запустите скрипт оптимизации снова

## Альтернативный способ (без Node.js)

Если у вас нет Node.js, вы можете использовать онлайн инструменты:
1. https://squoosh.app/ - конвертация в WebP
2. https://imagemagick.org/ - командная строка
3. https://www.photopea.com/ - онлайн редактор

Конвертируйте изображения вручную и сохраните их с теми же именами, которые указаны в HTML.
